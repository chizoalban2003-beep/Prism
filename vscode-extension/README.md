## PRISM Assistant for VS Code

This extension connects VS Code to your locally-running PRISM AI assistant (127.0.0.1:8743), giving you inline code explanation, file review, error fixing, and a chat panel — all powered by the same personal model that knows your values and preferences.

To install, open VS Code, go to Extensions → "…" menu → "Install from VSIX" (if you've packaged it) or use **Developer: Install Extension from Location** to load the `vscode-extension/` folder directly as an unpacked extension.
